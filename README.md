# CheapoJoe

Compiling info about nearby yard sales.

## What Can This Do?
Nothing yet, but here's what it'll hopefully be able to do:

```
# List nearby yard sales
./cheapojoe zip=27703

# Optional args
date "2018-03-24"
zip "12345"
limit 3
```
